Perfil

Desarrollador full-stack con foco en software de negocio real.

En Consultoría LecarTI desarrollo, mantengo y evoluciono sistemas de negocio en producción — ERPs, hotelería y facturación — trabajando el ciclo completo: desde el modelo de datos en SQL Server hasta la interfaz. Mi especialidad es el ecosistema C#/.NET (ASP.NET Core, WinForms, Entity Framework, Dapper) con fuerte dominio de facturación electrónica CFDI 4.0 e integración con el SAT.

Reescribí desde cero MotelWeb, migrando un sistema heredado en Visual Basic 6 a una web moderna que corre en la nube o localmente. He diseñado APIs REST propias con CORS y documentación Swagger, montado pipelines de CI/CD a Azure, e integrado tiempo real (SignalR), biometría y servicios de terceros. También desarrollo móvil con Flutter bajo Clean Architecture.
